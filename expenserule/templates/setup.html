ExpenseRule Automated receipt categorization for freelancers
●  Privacy & Data Notice

ExpenseRule processes your receipts entirely on this machine. Only the image content of each receipt is sent to the OpenAI API (GPT-4o-mini) to extract merchant name, date, and total amount. No receipt data is retained on OpenAI’s servers beyond the duration of each API call.

Your expense records are stored in a SQLite database at ~/.expenserule/expenses.db. Your API key is stored at ~/.expenserule/openai_api_key with 600 permissions — readable only by your user account.
