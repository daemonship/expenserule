Welcome to ExpenseRule

Privacy Notice

ExpenseRule processes your receipts locally. Only the image content of each receipt is sent to the OpenAI API (GPT-4o-mini) to extract merchant name, date, and amount. No receipt data is stored on OpenAI's servers beyond the duration of the API call. Your expense records are stored in a SQLite database on this machine only.

Your OpenAI API key is stored in ~/.expenserule/openai_api_key with 600 file permissions (readable only by you).
